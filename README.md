
# BPLab LIMS V4.3 全过程示意版

以原V4企业UI为基础，演示以下完整流程：

- 样品入库、登记和检测任务自动创建
- 所有人查看样品当前状态、位置、负责人及完整轨迹
- 任务分配、实验员领取样品、检测、提交复核
- 待复核列表立即刷新
- 复核通过后记录锁定
- 样品归还、样品管理员确认回库、留样保存
- 独立原始记录表下载中心
- 修改追踪采用列表、时间线、修改前后对照
- 修改后重新复核并生成新版Word
- 修改位置在新版Word中使用红色字体，追加修改说明页
- SOP和原始记录表版本管理
- 账号密码及五类角色权限
- 登录令牌有效期7天，刷新页面不会退出
- 删除定制式固定义齿与活动义齿两个视觉检测项目

## GitHub / Streamlit部署

仓库根目录应直接看到 `app.py`。Streamlit入口选择 `app.py`，Python建议3.12。

## 说明

这是流程和功能示意版，SQLite适合本地试运行。Streamlit Community Cloud重启时本地数据库可能重置。
正式上线应把数据库迁移至PostgreSQL/Supabase等持久数据库，并把登录令牌改为安全Cookie或OIDC。
粗糙度和厚度项目当前缺少可用的DOCX原始记录模板，系统会生成备用表；管理员上传正式DOCX后会自动使用正式模板。
