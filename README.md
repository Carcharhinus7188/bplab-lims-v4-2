# BPLab Trace V4.5.1 GitHub Package

大连标普实验室样品全过程追溯系统

## 本版新增

入库环节增加两个默认预设：

- 默认客户
  - 客户编号：`C-DEFAULT`
  - 客户名称：`默认客户`
- 默认样品
  - 样品编号：`S-DEFAULT`
  - 样品名称：`默认样品`
  - 默认单位：`件`

这样即使暂时没有正式客户信息库和样品名称库，也可以先完成样品入库。

管理员和收样员仍然可以在“基础资料”中继续新增正式客户和正式样品名称。

## 继续保留的功能

- 所有业务时间使用中国大陆时区 `Asia/Shanghai（UTC+8）`
- 管理员和收样员可维护客户与样品名称
- 只有管理员可以上传SOP及实验原始记录表
- 计划完成日期默认按接收日期顺延一个自然月
- 样品编号自动生成并允许手工修改
- A区域、B区域入库位置
- 任务提醒、实验员接收、时间轴、复核、回库和修改追踪

## GitHub上传

解压后，将文件夹中的全部内容上传到GitHub仓库根目录。

Streamlit入口文件：

`app.py`

上传后建议执行：

`Manage app -> Reboot app`
