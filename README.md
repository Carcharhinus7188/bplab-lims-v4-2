# BPLab Trace V5.5 受控原始记录模板驱动版

本版以用户提供的“新版SOP文件”压缩包中的10份原始记录表为唯一正式模板来源。

## 核心更新

### 1. 原始记录按原模板直接填写

- 不再重新设计或生成通用原始记录表；
- 不新增、删除、合并、拆分模板表格；
- 不增加设备附表、附件附表或修改附页；
- 程序读取模板已有可填写单元格，实验员在实验记录阶段逐项填写；
- 已知委托、样品、材料、检测方法、人员、地点和设备快照信息自动带入；
- 模板要求但系统无法自动确定的数据保持未完成状态，必须由实验员填写；
- 不适用项目必须明确填写“不适用”；
- 复选框项目必须至少选择一项并将`□`改为`☑`；
- 字段未完成时只能保存草稿，禁止提交复核；
- 下载时只把内容写回原模板原有位置。

模板原文件SHA-256与结构信息见：

- `template_hash_manifest.json`
- `template_hash_manifest.csv`
- `template_field_manifest.csv`（全部模板可填写位置清单）

### 2. 附件追溯与正式原始记录分离

附件不写入正式原始记录表。附件追溯暂不设置“软件或设备”字段，设备信息由实验配置快照和正式原始记录管理。

附件索引保留：委托、任务包、任务、样品、附件类型、文件名、相对路径、时间、上传人、说明、原始/处理关系和SHA-256。

### 3. 单据中心增加动态Excel下载

在选定委托后可下载：

`内部实验数据追溯工作簿.xlsx`

Excel按当前委托实时生成，包含：

- 使用说明；
- 实验总台账；
- 附件索引；
- 图片粘贴区；
- 修改记录；
- 复核记录；
- 每个实验的独立追溯工作表。

Excel不替代正式受控原始记录表。

### 4. 当前10份受控原始记录

- R001 表面粗糙度；
- R004 金属—陶瓷结合裂纹萌生；
- R005 金属内部质量X射线灰度分析；
- R006 翘曲变形；
- R007 热膨胀系数；
- R009 陶瓷牙耐急冷急热；
- R010 弯曲性能；
- R011 维氏硬度；
- R012 色稳定性；
- R013 厚度测量。
