# BPLab Trace V5.7 全实验完整映射版

大连标普检测有限公司实验全过程记录系统。

本版将“实验员简洁操作”“受控原始记录母版回填”“检验报告结果输出”和“内部附件追溯”分开处理。实验员只确认现场状态并填写真实原始测量值；前序数据、设备快照、固定参数、计算、判定和文件回填由系统完成。

## 已实现

- 10项实验的独立业务字段、默认参数、原始测量、计算、判定和报告结果规则
- 委托、生产单位、样品、编号、方法、地点、人员等前序信息只读继承
- 设备与实验动态配置，任务下发时冻结配置和设备快照
- 简洁的五步实验员界面，样品卡片录入，不展示Word表格或模板坐标
- 合理默认值、正常状态默认勾选、异常和偏离条件显示
- 7份DOCX受控原始记录母版原位回填，导出前后自动校验表格和页面结构
- 报告输出真实检验结果、标准要求、设备、环境、日期、样品说明和最终结论
- 禁止任何报告结果使用“详见原始记录”
- 全部记录复核且样品回库后自动建立待发布报告
- 首页、报告中心、单据中心进入时自动补建遗漏报告
- 附件追溯不含软件或设备字段，保存SHA-256校验值
- 单据中心下载内部实验数据追溯Excel
- 记录版本、复核、设备配置和报告状态审计留痕

## 本地启动

建议使用Python 3.12：

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

Windows激活虚拟环境时使用：

```bat
.venv\Scripts\activate
```

## 受控母版说明

当前资料包内有7份可直接无损回填的DOCX原始记录母版。表面粗糙度只有Pages母版；裂纹萌生和厚度测量缺少对应受控原始记录DOCX；正式检验报告母版也未包含在当前工作区。系统不会伪造受控文件，缺少母版时会阻止正式导出或发布，并明确列出缺口。

完整状态见 [docs/受控母版状态.md](docs/受控母版状态.md)。

## 云端数据注意事项

Streamlit Community Cloud本地SQLite和本地附件目录不应作为正式CMA记录的唯一长期存储。试运行时请及时下载Word和Excel归档；正式运行应将数据库和附件迁移到实验室受控服务器或持久化数据库/对象存储。
