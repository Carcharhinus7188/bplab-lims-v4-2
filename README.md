# BPLab Trace V5.2 实验名称与方法版

本版本依据最新确认的业务逻辑更新。

## 核心调整

- 系统业务界面只显示实验名称和检测方法。
- 入库、样品资料、任务包、原始记录、复核、附件和报告只显示：
  - 实验名称
  - 检测方法
- “实验名称＋检测方法”作为不可拆分的对应关系。
- 检测方法中不提供“其他方法”。
- 删除“金属-陶瓷结合三点弯曲试验”及其SOP、原始记录模板和入口。
- 任务编号改为任务包内顺序号：
  - `BP20260722001-P01-T01`
  - `BP20260722001-P01-T02`
- 任务编号采用任务包内顺序号，不包含实验项目标识。

## 固定实验地点

实验员领取任务包时，只能从以下地点选择：

- 化学室
- 无损检测室
- 性能检测室
- 显微检测室
- 制样室
- 外观检测室
- 样品室

入库人员不选择实验地点。实验地点由实验员确认领用时确定，并自动带入原始记录、附件和时间轴。

## 继续保留

- 同一委托统一选择生产单位或受委托生产企业
- 同一委托录入多个不同样品组
- `BP年月日001-01`实体样品编号
- 样品资料库不设置生产单位
- 多实验任务包一次领用、完成后一次归还
- 每个实验独立原始记录和独立复核
- 材料名称自动跟随任务
- 实验照片、电脑截图和原始数据附件追溯
- 样品回库确认
- 同一委托汇总一份检验报告
- 检测员、核验员、批准人三级报告流程
- 中国大陆时区 `Asia/Shanghai（UTC+8）`

## 部署

将解压后的全部文件上传到GitHub仓库根目录，Streamlit入口选择：

`app.py`

本版使用新的数据库文件：

`data/bplab_trace_v52.db`

上传后在Streamlit Cloud执行：

`Manage app -> Reboot app`


## 测试预设资料

首次运行自动建立：

- 委托客户：测试委托客户（预设）
- 生产单位：测试生产单位（预设）
- 样品：测试金属试样（预设）
  - 规格型号：25 mm×2 mm×2 mm
  - 材料名称：钴铬合金
  - 单位：件
  - 检测项目：表面粗糙度试验、弯曲性能试验、维氏硬度试验

预设仅用于流程测试，可在基础资料中修改、停用或新增正式资料。
