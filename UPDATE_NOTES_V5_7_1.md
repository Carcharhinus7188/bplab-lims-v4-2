# BPLab Trace V5.7.1 更新说明

本次为不改变系统骨架的小版本更新。

## 1. 逐实验检测位置

实验员接收任务包时，每个实验子任务分别显示检测位置选择框。不同实验可以选择不同地点，选择结果写入子任务及任务配置快照。历史任务继续保留当时选择的地点。

## 2. 检测项目全部手动选择

新建委托样品组与任务包下发页面不再自动勾选检测项目。样品管理员必须根据本次委托和本次下发范围手动选择。

## 3. 实验时间轴

实验开始和结束时间不再以文本框录入。实验员使用“记录实验开始时间”和“记录实验结束时间”按钮，系统自动按中国大陆时区记录，并写入任务审计时间轴及原始记录数据。

## 4. 实时计算与判定

样品卡片先读取本次输入的原始测量值，再立即计算平均值、派生结果及符合性。符合显示绿色提示，不符合显示红色提示，未完成原始数据时显示等待状态。

## 保持不变

- 委托、样品组、任务包、实验任务的编号命名法则；
- 各角色菜单和操作窗口；
- 六步实验记录流程；
- 10项实验及其业务字段；
- 原始记录与检验报告Word母版；
- 动态实验/设备配置与历史快照；
- 附件追溯、内部追溯Excel和报告发布流程。
